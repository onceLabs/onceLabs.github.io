Instance Method

activateAccessCard(registrationKey:timeout:)

Activate a registered FarpointeAccessCard for a given time interval in order to unlock a nearby reader

Declaration

func activateAccessCard(registrationKey: String, timeout: ActivationTime) async -> Result<Void, FarpointeError>

Parameters

registrationKey

the alphanumeric key associated with the FarpointeAccessCard

timeout

time interval until deactivateAccessCard(registrationKey:) will be called if no reader responds

Return Value

.success if a reader responds with a successful unlocking result or .failure(FarpointeError) if activation or unlocking fails

Discussion

The provided Access Card will be activated until the timeout from the provided ActivationTime parameter occurs, in that case deactivateAccessCard(registrationKey:) will be called internally

This device must be registered using registerUser(email:) or registerUser(phoneNumber:)

Note

The provided registration key must be associated with a valid registered FarpointeAccessCard
