Instance Property

accessCards

Published list of FarpointeAccessCards currently registered to this device

Declaration

var accessCards: Published<[FarpointeAccessCard]>.Publisher { get }

Discussion

Changes to this list occur when a FarpointeAccessCard is added, removed, or changed.

let cancellable = FarpointeSdk.shared.accessCards.sink { [weak self] cards in
    print("Access Cards updated: \(cards)")
}
