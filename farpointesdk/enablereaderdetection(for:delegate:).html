Instance Method

enableReaderDetection(for:delegate:)

Enable Reader Detection for the given Reader UUID

Declaration

func enableReaderDetection(for readerUuid: UUID, delegate: ReaderDetectorDelegate?) -> Result<Void, FarpointeError>

Parameters

readerUuid

the associated reader UUID to detect

delegate

register for any detection events

Return Value

.success if enabling detection succeeds or .failure(FarpointeError) if enabling detection fails

Discussion

Reader Detection allows this device to scan for nearby Readers associated with the provided UUID.

Once a Reader is found, the ReaderDetectorDelegate .onReaderNearby is invoked containing information about the Reader’s Beacon.

The delegate’s .onReaderNearby function will be invoked every 1 second while the Reader is within range of this device (as per Apple’s CoreLocation documentation)

Note

Background Mode: Location Updates
