Instance Method

registerDevice(phoneNumber:)

Register this device using a valid phone number to the LEGIC Connect Server

Declaration

func registerDevice(phoneNumber: String) async -> Result<Void, FarpointeError>

Parameters

phoneNumber

The phone number in the format of +15555555555 (regional code, no space, no dash)

Return Value

Void if .success or FarpointeError if .failure

Discussion

The user will receive a text containing the confirmation token which must be confirmed with confirmUserRegistration(confirmationToken:)

Registration is required for registering, activating, and deactivating FarpointeAccessCards.

Once a device is registered, the registered accessCards and associated data will persist in local storage until the application is deleted or unregisterCurrentUser() is called

Once the application is deleted or unregisterCurrentUser(), all data is lost. There is no way to recover lost FarpointeAccessCards once a device is unregistered

Note

Registration must be confirmed with confirmUserRegistration(confirmationToken:) after this function yields a success
