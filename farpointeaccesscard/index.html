FarpointeAccessCard

Access Card used to unlock a nearby Reader

Overview

To unlock a nearby Reader using a FarpointeAccessCard, pass the registrationKey of a registered card to the FarpointeSdk .activateAccessCard(registrationKey: timeout:) function
