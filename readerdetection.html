Reader Detection

Allows the user’s device to scan for nearby Readers

Overview

Reader Detection utilizes the CoreLocation framework to scan for nearby iBeacons inside readers. This allows us to implement a Touchless Activation feature by activating a FarpointeAccessCard once a reader is detected nearby.

Note

Background Mode: Location Updates

Note

Privacy - Location When In Use

See: Handling Location Updates

Enabling Reader Detection

Once the required permissions are set in the application’s .plist file, you can enable reader detection for a given reader by its UUID.

class ReaderDetectorHandler: ReaderDetectorDelegate {
    func readerDetector(onReaderNearby withBeacon: Beacon) {
        print("Reader detected nearby: \(withBeacon)")
        
        // Now is probably a good time to activate an access card...  
    }
}

let delegate = ReaderDetectorHandler()
let result = FarpointeSdk.shared.enableReaderDetection(for: readerUuid, delegate: delegate)
switch result {
case .success(_):
    print("Reader detection for reader with uuid \(readerUuid) enabled!")
case .failure(let error):
    print("Oops. Something went wrong when enabling reader detection. Heres the reason: \(error.description)")
}

Once a reader is detected, the ReaderDetectorDelegate.onReaderNearby method is invoked containing information about the reader’s Beacon.

The delegate’s .onReaderNearby function will be invoked every 1 second while the reader is within range of this device (as per Apple’s CoreLocation documentation).

Once enabled, reader detection can also be disabled

// Disable detection for a single reader
let result = FarpointeSdk.shared.disableReaderDetection(config: .single(uuid: readerUuid))
switch result {
case .success(_):
    print("Reader detection disabled for reader with uuid \(readerUuid)")
case .failure(let error):
    print("Oops. Something went wrong when disabling reader detection. Heres the reason: \(error.description)")
}

// Disable detection for a all readers previously enabled
let result = FarpointeSdk.shared.disableReaderDetection(config: .all)
switch result {
case .success(_):
    print("Reader detection disabled for all readers")
case .failure(let error):
    print("Oops. Something went wrong when disabling reader detection. Heres the reason: \(error.description)")
}
