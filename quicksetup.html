Quick Setup

Quick setup guide for initial configuration, registering an access card, and access card activation.

Setup

All interactions with the FarpointeSDK are through the shared instance of FarpointeSdk. The initial configuration process is as follows:

  1. Provide your Legic Connect Server credentials. FarpointeSdk.setup(configuration:)

  2. Register the user’s device. FarpointeSdk.registerDevice(email:) or FarpointeSdk.registerDevice(phoneNumber:)

  3. Confirm device registration using the confirmation token sent to the user via SMS or email. FarpointeSdk.confirmDeviceRegistration(confirmationToken:)

  4. All done. The device is now registered The library will now maintain a list of FarpointeAccessCards registered to this device

Once a device is registered, it can be unregistered by calling FarpointeSdk.unregisterDevice(:) or by uninstalling the app. Once either case occurs, all FarpointeAccessCards registered to this device will be deleted from local data. This data is not recoverable.

After initial configuration is complete, all that is required for subsequent app launches is 1. Provide your Legic Connect Server credentials. FarpointeSdk.setup(configuration:)

Registering an Access Card

In order for a FarpointeAccessCard to be activated, it must first be registered to the user’s device.

let result = await FarpointeSdk.shared.registerAccessCard(registrationKey: someAccessCardKey)
switch result {
case .success(_):
    print("Access Card successfully registered!")
case .failure(let error):
    print("Oops. Something went wrong during registration. Heres the reason: \(error.description)")
}

Accessing Access Cards

A @Published list of the device’s registered FarpointeAccessCards is available for collection on the accessCards property of FarpointeSdk

let cancellable = FarpointeSdk.shared.accessCards.sink { [weak self] cards in
    print("An access card was changed, added, or removed: \(cards)")
}

Activating an Access Card

To unlock a nearby reader, a registered FarpointeAccessCard can be activated for a certain amount of time until the reader acknowledges the interaction and responds with a result.

let activationTime = ActivationTime.thirtySeconds
let result = await FarpointeSdk.shared.activateAccessCard(registrationKey: someAccessCardKey, timeout: activationTime)
switch result {
case .success(_):
    print("The reader unlocked!")
case .failure(let error):
    print("Oops. Something went wrong during activation. Heres the reason: \(error.description)")
}

Note

If no reader responds within the provided ActivationTime, the function will return with a timeout error, and the access card will be deactivated.

A FarpointeAccessCard can also be deactivated while it is activated.

let result = FarpointeSdk.shared.deactivateAccessCard(registrationKey: someAccessCardKey)
switch result {
case .success(_):
    print("Card successfully deactivated")
case .failure(let error):
    print("Oops. Something went wrong during deactivation. Heres the reason: \(error.description)")
}

Note

Only one FarpointeAccessCard can be activated at a time. An error will return otherwise.
